# Security Policy

Si identificas una vulnerabilidad, no dudes en abrir un issue público.

Asimismo puedes escribir a: r.madrid@axxenconsulting.com

Incluye, cuando sea posible:

- Versión afectada.
- Pasos para reproducir.
- Impacto estimado.
- Evidencia técnica mínima.

Axxen Consulting revisará el reporte y determinará el tratamiento correspondiente.
